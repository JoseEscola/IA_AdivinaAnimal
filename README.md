# IA_AdivinaAnimal
Adivina el Animal
Es un juego donde la IA aprende a adivinar un animal que estás pensando, haciendo preguntas de sí o no, y va mejorando su conocimiento cada vez que juega.

Cómo funciona:
1. La IA tiene una base de conocimiento inicial (reglas como: ¿Si vuela, es un pájaro, si nada o es un pez?).
2. Te hace preguntas basadas en ese conocimiento.
3. Si no acierta, aprende un nuevo animal y una nueva regla.
4. Así, cada vez "sabe" más animales y reglas.

Ejemplo de ejecución:
Piensa en un animal...
¿Vuela? (s/n): n
¿Nada? (s/n): n
¿Es un perro? (s/n): n
¡Vaya! ¿Qué animal era?: gato
Dime una pregunta que distinga un gato de un perro: ¿Maúlla?
¡Gracias! Ahora lo sé para la próxima.


Nota: La próxima vez que juegues, usará esa nueva pregunta para distinguir entre gato y perro.
